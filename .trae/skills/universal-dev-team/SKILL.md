# Universal Dev Team

## Goal
在单次会话中完成需求分析、方案设计、编码实现、验证测试与交付说明。

## Invoke When
- 用户希望“一条龙落地”
- 任务跨需求、开发、测试多个阶段
- 需要减少来回沟通并快速交付

## Workflow
1. 提炼目标与约束
2. 设计可实施方案
3. 直接编码并自测
4. 执行质量检查并输出交付摘要

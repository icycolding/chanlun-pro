# GitHub Search & Discovery

## Goal
开发前快速找到可复用项目、实现范式与最佳实践，减少重复造轮子。

## Invoke When
- 需要寻找现成开源实现
- 评估某技术方向的成熟方案
- 希望先做竞品或样例调研

## Workflow
1. 定义检索关键词与筛选条件
2. 对比候选仓库质量与活跃度
3. 提炼可复用模式与风险点
4. 输出采用建议与落地步骤

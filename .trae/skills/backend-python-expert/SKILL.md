# Backend Python Expert

## Goal
提升 Python 服务端在架构、并发、性能与可维护性方面的质量。

## Invoke When
- 开发或改造 FastAPI 接口
- 处理异步任务、队列、IO 密集逻辑
- 排查性能瓶颈或稳定性问题

## Checklist
1. 校验接口契约与数据模型
2. 评估异步边界与阻塞调用
3. 优化数据库与缓存访问路径
4. 补齐监控、错误处理与测试
